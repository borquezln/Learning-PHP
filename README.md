## PHP For Beginners
The first series' idea is to take an approach to PHP with some of this topics:

### The Fundamentals
* PHP Tags and Variables
* Conditionals and Booleans
* Arrays and associative Arrays
* Functions, Filters and Lambda Functions
* Separate Logic From the Template

### Dynamic Web Applications
* Page Links and PHP Partials
* Superglobals and Current Page Styling
* PHP Router
* Create a MySQL Database
* PDO and PHP Database Class
* Environments and Configuration Flexibility
* SQL Injection Vulnerabilities Explained
